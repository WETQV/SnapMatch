# SnapMatch 1.0.4: первый публичный релиз

SnapMatch теперь доступен как публичный проект. Это desktop-приложение для управления Telegram-ботом с AI-моделями, MCP-инструментами, голосовой обработкой, историей сообщений и режимом секретаря.

![Главные настройки](screenshots/main-settings.png)

В релиз вошёл Windows-установщик через Inno Setup. Он ставит приложение как обычную desktop-программу, добавляет ярлыки и кладёт FFmpeg рядом с приложением, если FFmpeg доступен при сборке. Для Debian/Ubuntu добавлен `.deb`-сценарий сборки через GitHub Actions.

![MCP](screenshots/mcp-tab.png)

MCP в SnapMatch нужен для подключения внешних tools: локальных скриптов, интеграций и серверов, совместимых с Model Context Protocol. Сервер можно добавить через GUI, выбрать transport, задать команду запуска, рабочую папку и правила доступа.

![Секретарь](screenshots/secretary-tab.png)

Режим секретаря помогает обрабатывать входящие сообщения: собирать короткие серии сообщений, готовить черновики, просить подтверждение или отвечать автоматически в выбранных сценариях.

Обсудить проект, задать вопрос или связаться напрямую можно в Telegram-группе: [C0BETKA](https://t.me/C0BETKA).

Спасибо Павлу Дурову и команде Telegram за Bot API и экосистему, которая позволяет делать такие инструменты поверх Telegram.
